# Landing de Negocio con nombre "Hogar y esencia"

## Descripción

Este proyecto consiste en el desarrollo de una **Landing** para un negocio, diseñada con **HTML5** y **CSS**. El objetivo es presentar la información del negocio de manera clara, atractiva y organizada, utilizando las técnicas de maquetación **Flexbox** y **CSS Grid** para lograr un diseño moderno y adaptable a diferentes dispositivos.

La estructura principal del sitio incluye:

- **Header:** Contiene el logotipo y el menú de navegación.
- **Jumbotron:** Sección principal que presenta el negocio con un mensaje destacado y un llamado a la acción.
- **Secciones de contenido:** Información sobre los servicios o productos ofrecidos.
- **Footer:** Contiene información de contacto y derechos reservados.

## Tecnologías utilizadas

- HTML5
- CSS
- Flexbox
- CSS Grid

## Objetivo

Desarrollar una Landing con un diseño responsivo, aplicando Flexbox y CSS Grid para organizar los elementos de la interfaz.
